Hello everyone. Today, I am excited to demonstrate StudySync. Keeping track of course schedules and figuring out how to study effectively can be overwhelming. StudySync is a secure, personalized academic concierge that takes your raw syllabus and turns it into a detailed, customized study plan utilizing evidence-based cognitive strategies.

[0:20 — WHAT IT IS]
StudySync is built as a Concierge Agent using the Google Agent Development Kit version 2.0. Under the hood, it coordinates multiple specialized LLM agents and accesses custom study planning tools via a Model Context Protocol server.

[1:00 — ARCHITECTURE]
(show assets/architecture_diagram.png)
Let's walk through the architecture. When a user submits a query, it first goes through a Security Checkpoint node. This node scrubs personal student data like emails and phone numbers, and inspects the text for prompt injection attempts. If safe, it passes the query to our Orchestrator agent.
The Orchestrator coordinates two specialized sub-agents: the Syllabus Parser and the Schedule Generator. Both agents leverage our local MCP server to format calendars and recommend productivity strategies. If a study plan demands more than 10 hours a week, the workflow pauses at a Human Approval node for student consent before compiling the final dashboard.

[3:00 — SECURITY & MCP]
Our security node guarantees student privacy by redacting sensitive data before passing it to any LLMs. And our MCP server exposes local, specialized Python functions directly to the agents, including custom calendar calculation and subject-based study tips.
